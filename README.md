*me-cms-link-scanner* plugin allows you to manage LinkScanner with
[MeCms platform](//github.com/mirko-pagliai/cakephp-for-mecms).

To install:

    $ composer require --prefer-dist mirko-pagliai/me-cms-link-scanner
